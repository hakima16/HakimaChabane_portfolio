Project Description – Customer Insights Dashboard

About the Project

The Customer Insights Dashboard is a business analysis project designed to help financial institutions better understand customer behavior and branch performance. The dashboard concept focuses on organizing key customer and product data into a clear, visual format that supports decision-making.

The goal of this project is to demonstrate how a Business Analyst can gather requirements, design data structures, and create meaningful insights that help teams improve customer engagement and operational efficiency.

Key Features

  • Daily and weekly customer activity tracking
  • Account openings (checking, savings, credit cards)
  • Referral performance
  • Customer demographics and trends
  • Branch-level performance comparison
  • Simple visual charts and summaries

Tech Stack Used

  • Excel – Data cleaning, formatting, and initial analysis.
  • Power BI – Dashboard creation, KPIs, charts, and interactive insights.
  • Business Analysis Techniques – Requirements gathering, user stories, and acceptance criteria.
  • Wireframing Tools – Simple mockups for dashboard layout and user interface.

BA Deliverables Included

  • Project overview and business problem statement
  • Requirements documentation
  • Sample data models
  • User stories and acceptance criteria
  • Wireframes for dashboard layout
