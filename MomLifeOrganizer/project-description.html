Project Description – MomLife Organizer

About the Project

MomLife Organizer is a small MERN application designed to help busy moms manage their family schedules in one place. The app allows the user to keep track of kids’ weekly activities, doctor appointments, shopping lists, cleaning tasks, and family trips. The goal is to organize multiple parts of family life through a simple, user-friendly interface.

The application uses a single login (web215user / LetMeIn!) to access protected pages. Once logged in, the user can view and manage all family-related data.

Data Structure & Relationships

The project includes sample data such as kids, activities, appointments, and tasks. These demonstrate a many-to-many relationship, for example:

  • One kid can have many activities
  • One activity can belong to multiple kids

The final application will support full CRUD (Create, Read, Update, Delete) on all major data types.

Tech Stack Used

  • MongoDB – Cloud database hosted on MongoDB Atlas.
  • Express.js – Backend API and routing.
  • React – Frontend user interface.
  • Node.js – Server environment for backend logic.
  • Netlify – Hosting for the frontend.
  • Render – Hosting for the backend API.

Deployment Plan
