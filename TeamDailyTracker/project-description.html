Project Description – Team Daily Tracker

About the Project

The Team Daily Tracker is a simple web application that allows users to create an account, log in, and track their daily work activities. The application includes a user table with a unique user ID and password, and anyone can create a new user account through the registration page.

After logging in, each user can perform full CRUD operations (Create, Read, Update, Delete) on their own daily activity entries. Each entry includes:

  • Checking accounts opened
  • Savings accounts opened
  • Credit cards
  • CCGs
  • Referrals
  • Calls

Users can also read entries created by other users, allowing team members to view each other's daily performance. However, users can only create, update, and delete their own entries. They cannot modify entries created by others. This ensures proper access control while still supporting team visibility.

Pages Included in the Project

  • Create Account (registration)
  • Login
  • Dashboard (view personal and team entries)
  • Add Entry
  • Edit Entry
  • Delete Entry
  • Logout

Tech Stack Used

  • PHP – Handles backend logic, user authentication, and CRUD operations.
  • MySQL – Stores user accounts, daily entries, and activity data.
  • HTML & CSS – Provides structure and styling for all pages.
  • JavaScript – Enhances user experience with validation and dynamic UI behavior.
